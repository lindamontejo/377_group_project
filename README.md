# 377_group_project

Exploring the sexual assault and sexual offenses data provided by CTA through the Chicago Data Portal 



Scripts: 

01_cleaning_data.r: This script contains an initial data exploration and transformation. 

02_eda.r: In this script different variables were explored further. Graphs were also made for easier analysis.

03_fact_checking_r: This script fact-checks some of the facts present in the presentation and story.


Data folder
This folder contains the original sexual assaults and offenses datasets, along with the original datasets. Different tables were 
saved to share with others or to make charts in Flourish.
